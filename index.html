what platforms does fourclient support?

fourclient is distributed as a cross-platform client for windows, linux and macOS (native builds). platform-specific installers and portable archives are provided when available.

how do i update fourclient?

updates are published on the gitHub releases page. you can enable automatic update checks in future versions; for now, download the latest release and replace the executable.

can i request features or report bugs?

yes — open an issue on the gitHub repo or join the discord community via the discord button above. provide logs and steps to reproduce when reporting bugs.
